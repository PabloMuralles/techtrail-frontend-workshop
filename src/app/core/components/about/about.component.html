What the hell means Frontend

Talking about Software development, the frontend is everything with which the user interacts.
From a user standpoint s Synonymous with the User Interface but, from a developer standpoint, it is the interface design and the programming that makes the interface function. Conversely, the backend processes the data, those actions take place behind scenes.
